تطوير روبوتات الدردشة (Bots)

الوصف

يتضمن هذا المشروع بناء ودمج روبوتات محادثة ذكية (Chatbots) قادرة على فهم اللغة الطبيعية وأتمتة المهام. تم العمل على نماذج مختلفة، بدءًا من الحلول المخصصة باستخدام منصة Rasa مفتوحة المصدر، وصولًا إلى دمج نماذج لغوية كبيرة (LLMs) مثل DeepSeek و Llama لإنشاء تجارب محادثة متقدمة.

الميزات الرئيسية

دمج النماذج اللغوية (LLMs)

ربط نماذج لغوية متقدمة مثل DeepSeek و Llama لتمكين الروبوتات من إجراء محادثات معقدة وتوليد استجابات دقيقة.

منصة Rasa

استخدام منصة Rasa لبناء روبوتات محادثة مخصصة مع إدارة كاملة للحوار وفهم النوايا (Intents) والكيانات (Entities).

واجهات أمامية (Frontend)

تطوير واجهات مستخدم بسيطة لعرض قدرات الروبوتات وتسهيل التفاعل معها عبر واجهات برمجية (APIs).

التقنيات المستخدمة

  • Python
  • Large Language Models (LLMs)
  • Rasa Open Source
  • API Integration
  • Natural Language Processing (NLP)
